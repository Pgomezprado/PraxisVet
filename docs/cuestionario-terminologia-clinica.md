# Cuestionario de terminología clínica — PraxisVet

**Reunión:** Martes 21 de abril de 2026
**Objetivo:** Cerrar el vocabulario oficial del producto para que todo el copy, formularios, reportes y PDFs hablen el mismo idioma que la clínica.
**Duración estimada:** 30 minutos
**Formato sugerido:** Conversación mientras muestran el sistema que usan hoy. Grabar si es posible.

---

## Cómo usar este documento

- No es un formulario para que ellos llenen. Es una **guía de preguntas** para ti.
- Pregunta mientras observas el flujo real.
- Anota las palabras literales que usan, incluso si suenan informales.
- Si dudan entre dos términos, pregunta cuál usan **con los tutores** y cuál **entre colegas** — pueden ser distintos.

---

## 1. Especies

- [x] Confirmado en reunión anterior: `canino`, `felino`, `exótico`
- [ ] Dentro de **exótico**, ¿necesitan subcategorías?
  - Aves
  - Reptiles
  - Roedores (cobayas, hámsteres, ratas)
  - Lagomorfos (conejos)
  - Hurones
  - Peces
  - Otros
- [ ] ¿Atienden alguna especie que no encaja acá? (equinos, bovinos, fauna silvestre, etc.)
- [ ] ¿Quieren que el sistema pida **raza** aparte o basta con la especie?

## 2. El animal

- [ ] ¿Lo llaman **paciente**, **mascota** o **animal**?
- [ ] Título de la ficha: ¿"Ficha del paciente", "Ficha clínica", "Historia clínica"?
- [ ] ¿Usan "**ingreso**" cuando llega uno nuevo, o algo distinto?

## 3. La persona responsable

- [ ] ¿**Tutor**, **propietario**, **dueño** o **responsable**?
- [ ] En Chile legal: "tenedor responsable". ¿Lo usan o suena burocrático?
- [ ] Cuando hablan con el tutor directamente, ¿tutean o tratan de usted?

## 4. La visita

- [ ] ¿**Consulta**, **atención**, **visita** o **cita**?
- [ ] ¿Diferencian **cita** (reserva en agenda) de **consulta** (atención realizada)?
- [ ] Cuando llega sin cita: ¿**urgencia**, **walk-in**, **sin cita**, **emergencia**?
- [ ] ¿Tienen algo como **control**, **seguimiento** o **recontrol**?

## 5. Procedimientos y tratamientos

- [ ] ¿**Procedimiento**, **intervención**, **tratamiento** o **servicio**?
- [ ] Cirugía: ¿**operación** o **cirugía**?
- [ ] Vacunas: ¿**vacunación**, **inmunización** o **plan sanitario**?
- [ ] Desparasitación: ¿término específico?
- [ ] Esterilización: ¿**castración**, **esterilización**, **ovariectomía**?
- [ ] **Sexo del paciente**: hoy el sistema dice solo "Macho / Hembra". ¿Necesitan distinguir **macho entero / macho castrado / hembra entera / hembra esterilizada**? (Clínicamente es relevante para dosis y diagnóstico)

## 6. Documentos clínicos

- [ ] ¿**Receta** o **prescripción**?
- [ ] En Chile hay **receta retenida** obligatoria para ciertos fármacos — ¿la llaman así?
- [ ] ¿**Certificado de salud**, **pasaporte sanitario**, **carnet**?
- [ ] Informe de alta: ¿**alta clínica**, **epicrisis**, **informe final**?
- [ ] ¿Qué documentos imprimen/entregan al tutor hoy? (Esto es oro — pide ver ejemplos reales)

## 7. Personal interno (títulos en el software)

- [ ] Veterinario/a: ¿**Veterinario**, **Médico veterinario**, **Dr./Dra.**?
- [ ] Recepción: ¿**Recepcionista**, **Secretario/a**, **Asistente de admisión**?
- [ ] Peluquería: ¿**Peluquero canino**, **Groomer**, **Estilista**?
- [ ] ¿Hay **auxiliar / técnico veterinario / TENS** que deba aparecer como rol aparte?
- [ ] ¿Cómo firman los documentos? (título + nombre, solo nombre, con RUT, con código de colegio?)

## 8. Servicios conexos

- [ ] Hospitalización: ¿**hospitalizado**, **internado**, **en observación**?
- [ ] Peluquería: confirmar que es flujo paralelo (no servicio extra). ¿Qué copy usan? "baño y peluquería", "estética canina", etc.
- [ ] ¿Tienen **laboratorio** propio o lo derivan? ¿Cómo lo registran?
- [ ] ¿**Imagenología** / **ecografía** / **radiografía** son módulos que quieren ver separados?

## 9. Dinero y documentos de pago

- [ ] ¿**Cobro**, **pago**, **cuenta**, **boleta**?
- [ ] ¿**Presupuesto** o **cotización**?
- [ ] Boleta vs factura: ¿cuándo emiten cada una? (dato para flujo SII)
- [ ] ¿Manejan **abonos / pagos parciales**?

## 10. Comunicación automatizada al tutor

- [ ] Recordatorios (cita, vacuna, control): ¿tono cercano, formal o clínico?
- [ ] ¿Por qué canal? (WhatsApp, SMS, email) — y sobre todo: ¿cuál **funciona**?
- [ ] ¿Quieren firmar los mensajes con el nombre de la clínica o del veterinario tratante?
